# <center>horseloft framework</center>

A simple php API framework

基于Swoole的API框架

当前版本的框架并没有使用Swoole的协程功能；

之所以使用Swoole为基础开发一个API框架，主要是因为它可以实现很多PHP无法实现的功能；
